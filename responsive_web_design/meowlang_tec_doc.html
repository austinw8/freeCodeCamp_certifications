Introduction

MeowLang is a playful, whimsical programming language inspired by the unpredictable nature of cats.

With MeowLang, code is written using feline-inspired syntax that mimics the mysterious and capricious behavior of cats. It’s designed to be fun and a bit chaotic, just like living with a cat, while still being functional for basic programming tasks.

This tec-doc will cover:

  • Cat variables
  • Whisker functions
  • Pounce operators
  • Cat loops
  • Cat conditionals
Cat Variables

In MeowLang, variables are declared by "feeding" them a value. Much like a cat, they can wander off and change their value at will, so don't be surprised when they decide to change unexpectedly.

The "feed" keyword is used to assign values, and you can "purr" to print out the current value.

feed food = "Fish"
purr food
Whisker Functions

Functions in MeowLang are called "whiskers," symbolizing the delicate balance between curiosity and chaos.

Functions can return a value, but expect them to occasionally misbehave, much like how a cat might choose to ignore you when you need them most.

whisker greet(name) {
feed greeting = "Hello, " + name
purr greeting
}
whisker greet("Fluffy")
Pounce Operator

Pouncing is a core concept in MeowLang, where one action triggers another—much like how a cat pounces on an unsuspecting object.

The "pounce" operator allows you to perform chained actions, with each pounce producing a consequence. Think of it as combining functions in a single line.

pounce feed toy = "Ball" purr toy
Cat Loops

Cats love to chase, and in MeowLang, loops simulate that endless pursuit.

The "chase" keyword creates a loop that will continue running until a certain condition is met. Just like a cat chasing something, it may never stop unless you intervene.

chase feed counter = 0 until counter > 5 {
feed counter = counter + 1
purr counter
}
Cat Conditionals

MeowLang has conditionals that act like a cat's judgment call—purring when something is good and swatting when it's bad.

The "swat" keyword is used for an else statement, while the "purr" keyword is used when conditions are met. Cats are notoriously selective, so be prepared for unexpected results.

feed mood = "Hungry"
if mood == "Hungry" {
purr "Feed me, human!"
} swat {
purr "I'll ignore you now."
}
